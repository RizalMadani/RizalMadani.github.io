SME Branding

Tahun
2020 - 2021
Jenis
Web App
Role
Full-stack Dev
Teknologi Utama:
  • CodeIgniter 3
  • MySQL
  • HTML-CSS-JS
  • Bootstrap

SME Branding adalah sebuah platform layanan branding untuk UMKM dengan layanan utamanya adalah desain kemasan dan logo. SME Branding merupakan salah satu bagian dari layanan-layanan SMEsPedia yang disediakan untuk para UMKM.

Proyek ini direncanakan sebagai sebuah platform untuk menunjang kegiatan layanan SME Branding. Layanan tersebut akan digunakan oleh UMKM yang akan memesan layanan, freelancer sebagai pelaku layanan, dan admin yang mengelola kegiatan layanan dari pemesanan sampai selesai.

Authentication & Authorization

Fitur autentikasi dan autorisasi ini digunakan untuk mengelola pengguna yang masuk ke platform dan membatasi akses sesuai role pengguna.

Pembuatan fitur ini dilakukan secara manual dengan mengikuti panduan yang disarankan karena CodeIgniter 3 tidak mempunyai fitur ini secara bawaan.

Aplikasi Dasbor

Untuk memenuhi kebutuhan pengguna maka dibuatkan sebuah aplikasi seperti aplikasi dasbor admin. Konten dan alur aplikasi dibuat menyesuaikan kebutuhan kegiatan masing-masing role pengguna.

Melalui aplikasi ini dimungkinkan untuk melakukan CRUD data dan menjalankan alur bisnis layanan.

SIKOST

Tahun
2020
Jenis
Web App
Role
Back-end Dev
Teknologi Utama:
  • CodeIgniter 3
  • MySQL
  • HTML-CSS-JS

Sikost adalah sebuah platform yang digunakan untuk mencari kost atau mengelola kost. Ada tigas jenis role pengguna, yaitu: customer, pengelola kost, dan admin.

Proyek ini dibangun menggunakan CodeIgniter versi 3 yaitu sebuah framework PHP dan MySQL untuk manajemen database. Kombinasi PHP+MySQL merupakan tech stack yang umum digunakan untuk membangun sebuah web.

Dasbor Admin

Admin aplikasi bisa mengelola konten web menggunakan dasbor ini. Di sini juga ditampilkan beberapa informasi seperti jumlah user, jumlah kost, dan lain-lain.

Beberapa kegiatan admin yang bisa dilakukan dengan aplikasi ini yaitu mengelola user, mengelola kost, mengelola artikel, dan juga memverifikasi penglola kost.

Pengelolaan Kost

Aplikasi memiliki fitur untuk mengelola kost dan juga hal lain seperti mengelola kamar, mengelola tagihan, mengelola transaksi, dan lain-lain. Fitur-fitur tersebut diperuntukan untuk pengguna dengan role Pengelola Kost.

Monitor Tagihan Kost

Bagi customer bisa memantau dan mengetahui status tagihan kostnya melalui aplikasi ini. Customer juga bisa mengirimkan bukti bayar melalui aplikasi agar memudahkan Pengelola kost.
